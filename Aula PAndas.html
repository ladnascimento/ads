Biblioteca Pandas

Nesta aula, vamos falar sobre a biblioteca Pandas, uma das principais bibliotecas de análise de dados em Python.

Introdução ao Pandas

O Pandas é uma biblioteca Python para manipulação e análise de dados. Ele fornece estruturas de dados de alto desempenho e fáceis de usar, bem como ferramentas para trabalhar com dados de diversas fontes, como arquivos CSV, Excel, bancos de dados SQL e até mesmo páginas da web.

Instalação do Pandas

Antes de começarmos a usar o Pandas, precisamos instalá-lo. A maneira mais fácil de instalar o Pandas é usando o pip, o gerenciador de pacotes padrão do Python. Para instalar o Pandas usando o pip, basta abrir um terminal ou prompt de comando e digitar o seguinte comando:

pip install pandas

Estruturas de dados no Pandas

Séries

Uma série é uma estrutura de dados unidimensional semelhante a um array que pode conter qualquer tipo de dado. Cada elemento da série é indexado com um rótulo, que pode ser um valor numérico ou um rótulo de texto. Você pode criar uma série a partir de uma lista, tupla, dicionário ou de um array NumPy.

import pandas as pd
import numpy as np

# Criando uma série a partir de uma lista
s = pd.Series([1, 3, 5, np.nan, 6, 8])

print(s)

DataFrames

Um DataFrame é uma estrutura de dados bidimensional semelhante a uma tabela de banco de dados, em que cada coluna pode ter um tipo de dado diferente. Ele pode ser criado a partir de uma lista de listas, uma série de dicionários, um array NumPy ou a partir de arquivos CSV, Excel, SQL, entre outros.

import pandas as pd
import numpy as np

# Criando um DataFrame a partir de um dicionário
data = {'Nome': ['João', 'Maria', 'Pedro', 'Lucas'],
        'Idade': [20, 25, 30, 35],
        'Altura': [1.75, 1.68, 1.80, 1.82]}
df = pd.DataFrame(data)

print(df)

Leitura de dados com Pandas

Uma das principais tarefas do Pandas é ler dados de diferentes fontes. O método mais usado para ler dados em Pandas é o método read(), que permite ler dados de diversos formatos, incluindo CSV, Excel, SQL, JSON e HTML. Abaixo estão alguns exemplos:

CSV

Para ler um arquivo CSV, basta usar o método read_csv(). Por exemplo:

import pandas as pd
df = pd.read_csv('dados.csv')

Excel

Para ler um arquivo Excel, basta usar o método read_excel(). Por exemplo:

import pandas as pd
df = pd.read_excel('dados.xlsx')

SQL

Para ler dados de um banco de dados SQL, basta usar o método read_sql(). Por exemplo:

import pandas as pd
import sqlite3
conn = sqlite3.connect('banco.db')
df = pd.read_sql('SELECT * FROM tabela', conn)

JSON

Para ler um arquivo JSON, basta usar o método read_json(). Por exemplo:

import pandas as pd
df = pd.read_json('dados.json')

HTML

Para ler uma tabela HTML em uma página da web, basta usar o método read_html(). Por exemplo:

import pandas as pd
dfs = pd.read_html('http://pagina.com/tabela.html')

Observe que o método read_html() retorna uma lista de DataFrames, já que uma página da web pode conter várias tabelas.

O método read()

O método read() é uma forma conveniente de ler dados de diferentes fontes e criar um DataFrame do Pandas. Ele tem muitos parâmetros opcionais que permitem personalizar a forma como os dados são lidos e transformados em um DataFrame.

Veja abaixo um exemplo básico de uso do método read():

import pandas as pd
df = pd.read('dados.csv', sep=';', header=0)

Observe que o método read_html() retorna uma lista de DataFrames, já que uma página da web pode conter várias tabelas. É importante verificar o conteúdo dos DataFrames gerados, pois eles podem precisar de limpeza e transformação antes de serem usados para análise de dados.

Filtrando dados com Pandas

Uma das principais tarefas de análise de dados é filtrar os dados de acordo com critérios específicos. O Pandas oferece diversas ferramentas para filtragem de dados. Abaixo estão alguns exemplos:

Seleção por valor

Uma das formas mais simples de filtrar dados é selecionando os dados que possuem um valor específico em uma determinada coluna. Para isso, usamos o método loc(), que permite selecionar linhas e colunas pelo rótulo ou pela condição.

import pandas as pd
df = pd.read_csv('dados.csv')
filtro = df.loc[df['coluna'] == 'valor']

O exemplo acima seleciona todas as linhas do DataFrame onde o valor da coluna "coluna" é igual a "valor".

Seleção por intervalo

Também é possível selecionar os dados com base em um intervalo de valores. Para isso, usamos o método loc(), passando como argumento uma condição que define o intervalo.

import pandas as pd
df = pd.read_csv('dados.csv')
filtro = df.loc[(df['coluna'] > valor_minimo) & (df['coluna'] < valor_maximo)]

O exemplo acima seleciona todas as linhas do DataFrame onde o valor da coluna "coluna" está entre "valor_minimo" e "valor_maximo".

Seleção por string

Para selecionar as linhas do DataFrame onde uma determinada coluna contém uma string específica, usamos o método str.contains().

import pandas as pd
df = pd.read_csv('dados.csv')
filtro = df[df['coluna'].str.contains('string')]

O exemplo acima seleciona todas as linhas do DataFrame onde a coluna "coluna" contém a string "string".

Seleção por categorias

Para selecionar as linhas do DataFrame onde uma determinada coluna contém um valor de uma categoria específica, usamos o método isin().

import pandas as pd
df = pd.read_csv('dados.csv')
filtro = df[df['coluna'].isin(['categoria1', 'categoria2'])]

O exemplo acima filtra o DataFrame df para incluir apenas as linhas em que o valor da coluna idade é maior que 30 e o valor da coluna sexo é igual a "F".

Método plot() do Pandas

O método plot() do Pandas é uma ferramenta poderosa para visualização de dados em Python. Ele é construído sobre a biblioteca Matplotlib e oferece uma ampla variedade de gráficos e gráficos customizáveis para criar visualizações de dados de alta qualidade. Nesta aula, vamos abordar como utilizar o método plot() do Pandas.

Criando gráficos com o método plot()

Para criar um gráfico com o método plot(), é necessário ter um objeto Pandas que contenha os dados que você deseja visualizar. O método plot() pode ser chamado diretamente no objeto Pandas e, em seguida, especificando o tipo de gráfico que você deseja criar. Por exemplo:

import pandas as pd
import matplotlib.pyplot as plt

df = pd.DataFrame({'x': [1, 2, 3, 4, 5], 'y': [2, 4, 6, 8, 10]})
df.plot(kind='line', x='x', y='y')
plt.show()

O código acima cria um gráfico de linhas com os valores de x no eixo horizontal e os valores de y no eixo vertical. A função plot() é usada para criar o gráfico, enquanto a função show() é usada para exibir o gráfico.

Tipo de gráficos disponíveis

O método plot() oferece vários tipos de gráficos que podem ser criados especificando o parâmetro kind. Alguns dos tipos de gráficos mais comuns incluem:

  • line: Gráfico de linha.
  • bar: Gráfico de barras verticais.
  • barh: Gráfico de barras horizontais.
  • hist: Histograma.
  • pie: Gráfico de pizza.
  • scatter: Gráfico de dispersão.

Para criar um gráfico de barras verticais, podemos utilizar o seguinte código:

df.plot(kind='bar', x='x', y='y')
plt.show()

Customizando gráficos com o método plot()

O método plot() também oferece uma ampla variedade de opções de personalização de gráficos. Algumas das opções incluem a adição de títulos, rótulos de eixo, legendas e personalização de cores e estilos de linha. Abaixo estão alguns exemplos:

Biblioteca Matplotlib

A biblioteca Matplotlib é uma das bibliotecas mais populares para visualização de dados em Python. Ela oferece uma ampla variedade de gráficos e gráficos customizáveis para criar visualizações de dados de alta qualidade. Nesta aula, vamos abordar como utilizar a biblioteca Matplotlib.

Instalando o Matplotlib

Antes de começar a utilizar o Matplotlib, é necessário instalá-lo. Para instalar o Matplotlib, basta executar o seguinte comando no terminal:

pip install matplotlib

Criando gráficos com o Matplotlib

Para criar um gráfico com o Matplotlib, é necessário importar a biblioteca e chamar a função apropriada. Por exemplo, para criar um gráfico de linhas simples, podemos utilizar o seguinte código:

import matplotlib.pyplot as plt
x = [1, 2, 3, 4, 5]
y = [2, 4, 6, 8, 10]
plt.plot(x, y)
plt.show()

O código acima cria um gráfico de linhas com os valores de x no eixo horizontal e os valores de y no eixo vertical. A função plot() é usada para criar o gráfico, enquanto a função show() é usada para exibir o gráfico.

Customizando gráficos com o Matplotlib

O Matplotlib oferece uma ampla variedade de opções de personalização de gráficos. Algumas das opções incluem a adição de títulos, rótulos de eixo, legendas e personalização de cores e estilos de linha. Abaixo estão alguns exemplos:

Adicionando um título e rótulos de eixo

Para adicionar um título e rótulos de eixo a um gráfico, podemos utilizar as funções title(), xlabel() e ylabel(). Por exemplo:

import matplotlib.pyplot as plt
x = [1, 2, 3, 4, 5]
y = [2, 4, 6, 8, 10]
plt.plot(x, y)
plt.title('Meu primeiro gráfico')
plt.xlabel('Eixo X')
plt.ylabel('Eixo Y')
plt.show()

Biblioteca Seaborn

A biblioteca Seaborn é uma biblioteca de visualização de dados em Python baseada na biblioteca Matplotlib. Ela fornece uma interface de alto nível para criar visualizações estatísticas atraentes e informativas. Nesta aula, vamos abordar como utilizar a biblioteca Seaborn.

Instalando o Seaborn

Antes de começar a utilizar o Seaborn, é necessário instalar a biblioteca. A forma mais fácil de fazer isso é utilizando o gerenciador de pacotes pip. Para instalar o Seaborn, abra o terminal e digite o seguinte comando:

pip install seaborn

Criando gráficos com o Seaborn

Para criar um gráfico com o Seaborn, é necessário ter um objeto Pandas que contenha os dados que você deseja visualizar. A biblioteca Seaborn fornece várias funções para criar diferentes tipos de gráficos, como gráficos de barras, gráficos de dispersão, gráficos de caixa, entre outros.

Por exemplo, para criar um gráfico de dispersão com o Seaborn, podemos utilizar o seguinte código:

import seaborn as sns
import pandas as pd

df = pd.DataFrame({'x': [1, 2, 3, 4, 5], 'y': [2, 4, 6, 8, 10]})
sns.scatterplot(data=df, x='x', y='y')
plt.show()

O código acima cria um gráfico de dispersão com os valores de x no eixo horizontal e os valores de y no eixo vertical. A função scatterplot() é usada para criar o gráfico.

Customizando gráficos com o Seaborn

O Seaborn também oferece uma ampla variedade de opções de personalização de gráficos. Algumas das opções incluem a adição de títulos, rótulos de eixo, legendas e personalização de cores e estilos de linha.
